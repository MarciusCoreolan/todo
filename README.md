# Список дел
Приложение список дел

## Функционал
1) Вывод спписка задач,
2) Добавление новой задачи,
3) Поиск по подстроке,
4) Завершение задачи,
5) Удаление задачи.

## Для реализации проекта использовалось
html,
scss,
js,
redux,
react-redux,
redux-thunk,
normalize.css,
prettier.

## Запуск
Приложение запускается на 3000 порте.

### npm start
